# Termux Installation Guide.
1. `apt update && apt upgrade -y`
2. `pkg install git`
3. `git clone https://github.com/AbhiModszYT/Libpatcher.git`
4. `cd Libpatcher`
5. `ls`
6. `chmod +x dump.sh`
7. `./dump.sh`

Enjoy...🙂
